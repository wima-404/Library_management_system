Library Management System
Description: Le Library Management System est une application web conçue pour gérer efficacement une bibliothèque. Elle permet la gestion des livres, des auteurs, des utilisateurs, des emprunts et des réservations, tout en offrant une interface moderne et sécurisée.

Ce projet est idéal pour un projet universitaire ou pour pratiquer les concepts de Spring Boot, JPA/Hibernate, REST API et architecture MVC.
